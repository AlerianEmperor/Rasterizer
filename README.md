# C-Rasterizer
Rasterizer in C++ to study how Graphics Driver work without third party library. The result is simillar to images rendered by OpenGL or Vulkan!

#Features

- Rasterize line and Triangle.
- Graphics Pipeline.
- Perspective Projection.
- Camera Movement. 
- Texture Mapping.
- Gourad Shading.
